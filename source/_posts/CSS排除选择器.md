# 用法
```CSS
li:not(:first-child){
    color:#0C58A7;
}
```
# 浏览器支持
表格中的数字注明了完全支持该属性的首个浏览器版本。
![](https://bj.bcebos.com/baidu-rmb-video-cover-1/0c0e2f226955fd010a4f06b59ddae45d.png)

# 定义
```:not(selector)``` 选择器匹配非指定元素/选择器的每个元素。
